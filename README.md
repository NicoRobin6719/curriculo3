## Sobre o Projeto

Este projeto foi desenvolvido utilizando **Next.js**, um framework baseado em **React**, criado para a construção de aplicações web modernas, rápidas e escaláveis.

O projeto foi inicializado com o comando `create-next-app`, que gera automaticamente a estrutura básica necessária para o desenvolvimento, incluindo configuração de rotas, fontes otimizadas e suporte a TypeScript.

---

## Como o Projeto Funciona

- O servidor de desenvolvimento é iniciado com o comando:

npm run dev
